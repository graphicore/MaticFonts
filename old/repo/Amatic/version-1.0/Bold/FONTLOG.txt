Basic Font Information

Amatic is a simple but effective hand drawn webfont. Amatic can be used for titling and text. Amatic has been designed to be used freely across the internet by web browsers on desktop computers, laptops and mobile devices.


More specifically, this release supports the following 
Unicode ranges: Latin-1

For each style there are 4 Source files:

1. FONT-OTF.ufo Original Source Files in UFO format

1. FONT-kern.fea Source kerning file in Adobe OpenType Feature File format

3. Font-TTF.sfd TrueType sources that correspond to the unhinted TTF file

4. FONT-unhinted.ttf TrueType binary, unhinted
